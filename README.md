# Multimodal Math Mentor

## Description
The **Multimodal Math Mentor** is an innovative educational tool designed to assist students in learning mathematical concepts using multiple modes of representation. Through interactive exercises, visual aids, and textual explanations, the project aims to enhance understanding and retention of mathematical principles.

## Installation
To set up the Multimodal Math Mentor locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/pa7003/Multimodal_Math_Mentor.git
   ```
2. Navigate into the project directory:
   ```bash
   cd Multimodal_Math_Mentor
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```

## Usage
To run the project, use the following command:
```bash
npm start
```
Open your browser and navigate to `http://localhost:3000` to access the application. Follow the on-screen instructions to start exploring mathematical concepts.
