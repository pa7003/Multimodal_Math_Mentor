# Calculus Formulas

## Derivatives
- d/dx(x^n) = n*x^(n-1)
- d/dx(sin x) = cos x
- d/dx(cos x) = -sin x
- d/dx(e^x) = e^x
- d/dx(ln x) = 1/x

## Integrals
- ∫ x^n dx = (x^(n+1))/(n+1) + C
- ∫ 1/x dx = ln|x| + C
- ∫ e^x dx = e^x + C

## Limits
- lim (x->0) (sin x / x) = 1
- lim (x->0) (e^x - 1) / x = 1
